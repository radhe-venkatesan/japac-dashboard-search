Why This Exists

A bit of context before you start clicking around

The Problem

17 dashboards and no easy way to navigate them.

Each dashboard tracks different things, uses different terminology, and lives in a different part of the app. Finding a specific number meant opening multiple reports and hoping to land on the right one. For anyone new to the team, understanding what we track required sitting with someone for a walkthrough.

The Solution

One search bar. Every measure. Instant answers.

826 measures were extracted from all 17 dashboards to build this app. Type any keyword and get the measure name, its definition, which dashboard it belongs to, and a direct link to open it. No more hunting around.
