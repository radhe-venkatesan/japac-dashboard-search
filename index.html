Why these tools?
💬

Claude on its own Limited

Claude.ai can read, write and answer questions but it cannot touch files on your computer. You would have to copy and paste everything manually, back and forth.

+
🖥️

VS Code gives Claude "hands" Used here

VS Code is a free tool that sits on your desktop. With the Claude Code extension, Claude can directly open, read and edit your Excel files just like an analyst working live inside your folders.

↓
What Claude actually did
✦

Claude Code  ×  VS Code

Claude worked like a data analyst on your team. It read the Excel files and PDF exports, understood the existing layout and colour coding, then wrote and ran Python scripts to populate the repository. No manual data entry, no copy-paste errors, and no reformatting needed.
